The following questions are about data protection and privacy.

D.1: Search for and identify personal data

Can the organization generally identify all locations where personal data is stored across the enterprise, including on internal servers or cloud storage, as well as those hosted by any third-party providers?

Does the organization have:

1. The ability to locate all instances of personal data pertaining to a given data subject?

2. A formal process in place to search for personal data in a consistent and timely manner?

3. Technology in place for personnel to use a single search to return all instances of personal data for a given data subject?
